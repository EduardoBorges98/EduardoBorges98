## Projetos em destaque

### 🦴 [spring-skeleton](https://github.com/EduardoBorges98/springSkeleton)

> Esqueleto profissional e reutilizável para projetos Spring Boot — pronto para uso em qualquer empresa.

Template de referência que uso como ponto de partida em projetos corporativos. Construído com as melhores práticas do mercado: arquitetura em camadas, JWT stateless, cache Redis, migrations versionadas e testes com banco de dados real via Testcontainers.

**Tecnologias:** `Java 21` `Spring Boot 3.2` `Spring Security` `Spring Data JPA` `JWT` `PostgreSQL` `Redis` `Flyway` `MapStruct` `Lombok` `Testcontainers`

```
✔  Autenticação stateless com JWT (access token + refresh token)
✔  Cache Redis com @Cacheable, @CachePut e @CacheEvict
✔  Soft delete e auditoria automática via JPA (@CreatedBy, @LastModifiedDate)
✔  Migrations versionadas com Flyway
✔  Testes unitários (Mockito) e de integração (Testcontainers + PostgreSQL real)
✔  Swagger UI com autenticação Bearer token
✔  Tratamento de erros padronizado com GlobalExceptionHandler
✔  Profiles configurados: local, test e prod
```
